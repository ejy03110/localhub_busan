# LocalHub 부산 권역 MVP HTML Prototype

## 실행 방법

1. 압축을 해제합니다.
2. `index.html`을 브라우저에서 엽니다.
3. 보안 정책에 따른 파일 접근 문제를 피하려면 VSCode Live Server 사용을 권장합니다.

## 포함 화면

- `index.html`: 홈
- `explore.html`: 지역 지도 탐색
- `festival.html`: 축제 캘린더
- `board.html`: 게시판 목록
- `post-detail.html`: 게시글 상세
- `post-write.html`: 게시글 작성 및 수정

## 기능

- 반응형 UI
- localStorage 게시글 CRUD
- 제목 및 카테고리 검색
- 수정·삭제 비밀번호 확인
- 플로팅 챗봇 UI
- 지도·축제 캘린더 정적 프로토타입

기본 게시글 비밀번호는 `1234`입니다.
