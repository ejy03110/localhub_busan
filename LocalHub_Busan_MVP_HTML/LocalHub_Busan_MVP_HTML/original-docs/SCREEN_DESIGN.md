# LocalHub 화면 설계서

## 디자인 콘셉트

- 공공 서비스의 신뢰감과 여행 서비스의 친근함을 결합
- 네이비와 블루를 주색으로, 지역·환경 이미지를 위해 틸 색상 사용
- 카드형 정보 구조, 넓은 여백, 명확한 행동 버튼
- 모바일에서 목록과 챗봇을 우선적으로 사용할 수 있도록 반응형 구성

## 공통 컴포넌트

- 상단 내비게이션
- 페이지 제목 및 breadcrumb
- Primary / Secondary / Danger 버튼
- 카드, 패널, 알림 영역
- 플로팅 챗봇
- 모바일 반응형 레이아웃

## 프로토타입 동작

- 게시글 데이터는 `localStorage`의 `localhub-mvp-posts` 키에 저장
- 기본 수정·삭제 비밀번호: `1234`
- 검색은 제목과 카테고리 기준
- 챗봇은 키워드 기반 데모 응답
- 지도 및 달력은 실제 연동 전 정적 화면 설계
